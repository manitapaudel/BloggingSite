## General info
This is a simple project of a blogging portal which allows users to read the articles posted in the website. 
And after registering themselves, they can add posts of their own. 

## Technologies
Project is created with:
````````
* HTML5
* CSS 
* Bootstrap 4
* Laravel version6.3.0
* PHP version 7.3.6
* Visual studio code 
* Git Bash as the integrated terminal for Visual Studio code 
* XAMPP Apache server , MySQL server
````````

## Setup
This project is easy to setup.
```
*Open the project in a text editor.
*Run the Apache and MySQL server in XAMPP.
*Open up the integrated terminal and run the artisan command "php artisan serve" to open the application on the browser.
```

## Features
The project has following features:
`````
* Read the awesome blogs written by various people.
* Write blogs yourself in this website after logging in. You can write poetry, your own stories and many other things. Show your creativity.
* When you feel like you want to edit your articles, you can do so. And even delete some if you feel like it. You can always write some more when you get some inspiration.
* Navigate posts from the corresponding tags and vice versa.
`````
